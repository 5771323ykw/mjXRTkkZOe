# 中小企业设备管理系统 java275

## 项目概述

中小企业设备管理系统是一款基于Spring Boot和Vue技术的设备管理应用，旨在帮助中小企业高效管理设备与相关业务。

## 技术栈

- 后端技术：Spring Boot，MyBatis
- 前端技术：Vue，ElementUI，Axios，ECharts

## 系统角色

系统包含以下三种角色：
- 管理员
- 员工
- 用户

## 运行环境

- 开发工具：IDEA
- JDK版本：具体版本信息（请根据实际情况填写）
- 数据库：MySQL
- 项目构建工具：Maven

## 功能模块

### 管理后台

- 个人中心
  - 修改密码
  - 个人信息更新修改
- 员工管理
  - 新增，修改，删除员工信息
- 用户管理
  - 新增，修改，删除用户信息
- 设备信息管理
  - 新增，修改，删除设备信息
- 配件信息管理
  - 新增，修改，删除配件信息
- 设备购买管理
  - 删除，审核操作，统计报表分析
- 配件购买管理
  - 删除，审核操作，统计报表分析
- 设备点检管理
  - 删除点检信息
- 设备润滑管理
  - 删除润滑信息
- 设备改造管理
  - 删除改造信息
- 事务报警管理
  - 删除报警信息
- 设备类型管理
  - 新增，修改，删除设备类型
